# 待办事项应用程序

这是一个简单的命令行待办事项应用程序，使用C语言编写。

## 功能特点

- ✅ 添加新的待办事项
- 📋 查看所有待办事项列表
- ✅ 标记待办事项为已完成
- 🗑️ 删除不需要的待办事项
- 👋 简单友好的命令行界面

## 如何运行

### 编译程序

确保您的系统已安装C编译器（如GCC、Clang或MSVC）。

**使用GCC或Clang（Linux/macOS/WSL）：**
```bash
gcc -o todo_app todo_app.c
```

**使用MSVC（Windows命令提示符）：**
```cmd
cl todo_app.c /Fe:todo_app.exe
```

### 运行程序

编译完成后，运行生成的可执行文件：

**Linux/macOS/WSL：**
```bash
./todo_app
```

**Windows：**
```cmd
todo_app.exe
```

## 可用命令

1. **添加待办事项**
   ```
   add <标题> [描述]
   ```
   示例：
   ```
   add 完成Python作业 今天晚上之前完成
   ```

2. **查看所有待办事项**
   ```
   list
   ```

3. **标记待办事项为已完成**
   ```
   done <ID>
   ```
   示例：
   ```
   done 1
   ```

4. **删除待办事项**
   ```
   delete <ID>
   ```
   示例：
   ```
   delete 1
   ```

5. **退出应用**
   ```
   exit
   ```

## 注意事项

- 此应用程序的数据仅保存在内存中，关闭应用后所有数据将丢失
- 待办事项ID是自动生成的，从1开始递增
- 已完成的待办事项会显示✅标记，未完成的显示🔄标记
